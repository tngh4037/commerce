# spring-boot-study

# 소개
- Spring Boot 를 기반으로 학습 내용을 정리하기 위한 프로젝트입니다. ( 관리자 사이트 성격의 웹 애플리케이션 )
- 주로 아래와 같은 내용을 다루며, 점진적 리팩터링을 병행할 예정입니다.
    - 특정 기능을 구현하기 위한 다양한 개발 방법
    - 일관성 있는 프로젝트 구조
    - 예외 처리
    - 테스트 코드
    - ORM 전환
    - Security
