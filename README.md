# spring-boot-study

# 소개
- Spring Boot를 기반으로 학습 내용을 정리하기 위한 프로젝트입니다. ( 관리자 사이트 성격의 웹 애플리케이션 )
- 특정 기능을 구현하기 위한 다양한 개발 방법, 일관성 있는 프로젝트 구조, 예외처리, 테스트 코드, ORM 전환 등의 학습 내용에 대해서 주로 다룰 예정이고, 하나씩 점진적으로 적용하고 리팩토링할 예정입니다.
